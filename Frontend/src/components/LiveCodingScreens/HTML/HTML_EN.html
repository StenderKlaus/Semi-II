HTML Tags and Functions

Basic HTML

DOCTYPE Declaration (<!DOCTYPE>)

The <!DOCTYPE> declaration is used at the beginning of an HTML document to specify the document type. It helps the browser understand the version of HTML being used.

Example:

<!DOCTYPE html>

HTML Element (<html>)

The <html> element is the root element of an HTML document. It contains all the other elements on the page.

Example:

<html>
  <head>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Head Element (<head>)

The <head> element is used to define the head section of an HTML document. It contains meta-information about the document, such as the document title, character encoding, stylesheets, and scripts.

Example:

<head>
  <title>Document Title</title>
</head>

Title Element (<title>)

The <title> element is used to define the title of an HTML document. It is displayed in the browser's title bar or tab.

Example:

<title>Document Title</title>

Body Element (<body>)

The <body> element represents the content of an HTML document. It contains the visible content that is displayed in the browser window.

Example:

<body>
  <h1>Hello, World!</h1>
</body>

Heading Tags (<h1> to <h6>)

HTML provides several heading tags, ranging from <h1> to <h6>. These tags are used to define headings and subheadings on a page.

Example:

<h1>Main Heading</h1>
<h2>Subheading</h2>

Paragraphs (<p>)

The <p> tag is used to define a paragraph of text.

Example:

<p>This is a paragraph.</p>

Line Break (<br>)

The <br> tag is used to insert a line break within a paragraph or block-level element.

Example:

This is the first line.<br>This is the second line.

Horizontal Line (<hr>)

The <hr> tag is used to insert a horizontal line on the page, typically used to separate sections.

Example:

<p>This is some text.</p>
<hr>
<p>This is some more text.</p>

Comment (<!--...-->)

Comments are used to add notes or explanations to the HTML code. They are not displayed on the web page and are ignored by the browser.

Example:

<!-- This is a comment -->
<p>This is some text.</p>

Forms and Input

Textarea

The <textarea> tag is used to create a multi-line text input area.

Example:

<textarea rows="4" cols="50">Enter text here...</textarea>

Button

The <button> tag is used to create a clickable button.

Example:

<button type="button">Click Me</button>

Select

The <select> tag is used to create a dropdown list.

Example:

<select name="cars"> <option value="volvo">Volvo</option> <option value="bmw">BMW</option> <option value="audi">Audi</option> </select>

Optgroup

The <optgroup> tag is used to group related options in a dropdown list.

Example:

<select name="cars"> <optgroup label="Swedish Cars"> <option value="volvo">Volvo</option> <option value="saab">Saab</option> </optgroup> <optgroup label="German Cars"> <option value="audi">Audi</option> <option value="bmw">BMW</option> </optgroup> </select>

Option

The <option> tag is used to define an option in a dropdown list.

Example:

<select name="cars"> <option value="volvo">Volvo</option> <option value="bmw" selected>BMW</option> <option value="audi">Audi</option> </select>

Label

The <label> tag is used to define a label for an input element.

Example:

<label for="name">Name:</label> <input type="text" id="name" name="name">

Fieldset

The <fieldset> tag is used to group related form elements together.

Example:

<fieldset> <legend>Contact Information</legend> <label for="name">Name:</label> <input type="text" id="name" name="name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> </fieldset>

Legend

The <legend> tag is used to define a caption for the <fieldset> element.

Example:

<fieldset> <legend>Contact Information</legend> <label for="name">Name:</label> <input type="text" id="name" name="name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> </fieldset>

Datalist

The <datalist> tag is used to provide a predefined list of options for an input element.

Example:

<input list="browsers"> <datalist id="browsers"> <option value="Chrome"> <option value="Firefox"> <option value="Safari"> <option value="Opera"> <option value="Edge"> </datalist>

Output

The <output> tag is used to display the result of a calculation or user action.

Example:

<form oninput="result.value=parseInt(a.value)+parseInt(b.value)"> <input type="number" id="a" name="a"> + <input type="number" id="b" name="b"> = <output name="result" for="a b"></output> <input type="submit" value="Calculate"> </form>

Frames and iframes

The <frame> tag defines an individual frame within a frameset. It is used in conjunction with the <frameset> and <noframes> tags.

Example:

<frameset cols="25%,75%"> <frame src="menu.html"> <frame src="content.html"> <noframes> <p>This page requires a frames-enabled browser.</p> </noframes> </frameset>

The <iframe> tag is used to embed another HTML document within the current document. It allows you to display content from another source.

Example:

<iframe src="https://youtu.be/Htq2EUgcrKE"></iframe>

Images

The <img> tag is used to embed an image into an HTML page.

Example:

<img src="image.jpg" alt="Description of the image">

Image Map

The <map> tag is used in conjunction with the <area> tag to create clickable areas within an image.

Example:

<img src="image.jpg" alt="Image Map" usemap="#image-map"> <map name="image-map"> <area shape="rect" coords="0,0,100,100" href="page1.html" alt="Area 1"> <area shape="circle" coords="150,150,50" href="page2.html" alt="Area 2"> </map>

Canvas

The <canvas> tag is used to draw graphics, animations, or other visual images using JavaScript.

Example:

<canvas id="myCanvas" width="200" height="200"></canvas>

Figure and Figcaption

The <figure> and <figcaption> tags are used together to group and caption self-contained content, such as images or illustrations.

Example:

<figure> <img src="image.jpg" alt="Description of the image"> <figcaption>Caption for the image</figcaption> </figure>

Pictures and Source Sets

The <picture> tag allows developers to define multiple sources for an image, depending on the viewport or device.

Example:

<picture> <source media="(min-width: 768px)" srcset="large-image.jpg"> <source media="(min-width: 480px)" srcset="medium-image.jpg"> <img src="small-image.jpg" alt="Description of the image"> </picture>

SVG

The <svg> tag is used to embed scalable vector graphics in an HTML page.

Example:

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" fill="red" /> </svg>

Audio / Video

The <audio> tag is used to embed audio content in an HTML page.

Example:

<audio src="audio.mp3" controls> </audio>

Source

The <source> tag is used as a child of the <audio> or <video> tag to specify the media source.

Example:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> </audio>

Track

The <track> tag is used as a child of the <audio> or <video> tag to specify timed text tracks, such as captions or subtitles.

Example:

<video controls> <source src="video.mp4" type="video/mp4"> <track src="captions.vtt" kind="captions" label="English" srclang="en"> </video>

Video

The <video> tag is used to embed video content in an HTML page.

Example:

<video src="video.mp4" controls> </video>

Links

Anchor (Link)

The <a> tag is used to create a hyperlink, which allows users to navigate to another web page or a specific section within the same page.

Example:

<a href="https://www.example.com">Visit Example.com</a>

Link

The <link> tag is used to link external resources, such as stylesheets, to an HTML document.

Example:

<link rel="stylesheet" href="styles.css">

Navigation (Nav)

The <nav> tag is used to define a section of a document that contains navigation links.

Example:

<nav> <a href="home.html">Home</a> <a href="about.html">About</a> <a href="contact.html">Contact</a> </nav>

Lists

Unordered List (ul)

The <ul> tag is used to create an unordered list, where the list items are displayed with bullet points.

Example:

<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>

Ordered List (ol)

The <ol> tag is used to create an ordered list, where the list items are displayed with numbers or other ordered markers.

Example:

<ol> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ol>

List Item (li)

The <li> tag is used to define a list item within an unordered or ordered list.

Example:

<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>

Directory (dir)

The <dir> tag is a non-standard HTML tag used to create a directory list, which is a special type of unordered list.

Example:

<dir> <li>File 1</li> <li>File 2</li> <li>File 3</li> </dir>

Definition List (dl)

The <dl> tag is used to create a definition list, which consists of terms and their corresponding definitions.

Example:

<dl> <dt>Term 1</dt> <dd>Definition 1</dd> <dt>Term 2</dt> <dd>Definition 2</dd> </dl>

Definition Term (dt)

The <dt> tag is used to define a term within a definition list.

Example:

<dl> <dt>Term 1</dt> <dd>Definition 1</dd> <dt>Term 2</dt> <dd>Definition 2</dd> </dl>

Definition Description (dd)

The <dd> tag is used to define the description or definition of a term within a definition list.

Example:

<dl> <dt>Term 1</dt> <dd>Definition 1</dd> <dt>Term 2</dt> <dd>Definition 2</dd> </dl>

Tables

Table (table)

The <table> tag is used to create an HTML table.

Example:

<table> </table>

Caption (caption)

The <caption> tag is used to provide a caption or title for an HTML table.

Example:

<table> <caption>Table Caption</caption> </table>

Table Header Cell (th)

The <th> tag is used to define header cells in an HTML table.

Example:

<table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> </table>

Table Row (tr)

The <tr> tag is used to define rows in an HTML table.

Example:

<table> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> </table>

Table Data Cell (td)

The <td> tag is used to define data cells in an HTML table.

Example:

<table> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> <tr> <td>Row 2, Cell 1</td> <td>Row 2, Cell 2</td> </tr> </table>

Table Head (thead)

The <thead> tag is used to group the header content in an HTML table.

Example:

<table> <thead> <tr> <th>Header 1</th> <th>Header 2</th> </tr> </thead> </table>

Table Body (tbody)

The <tbody> tag is used to group the main content in an HTML table.

Example:

<table> <tbody> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> <tr> <td>Row 2, Cell 1</td> <td>Row 2, Cell 2</td> </tr> </tbody> </table>

Table Footer (tfoot)

The <tfoot> tag is used to group the footer content in an HTML table.

Example:

<table> <tfoot> <tr> <td>Footer 1</td> <td>Footer 2</td> </tr> </tfoot> </table>

Table Column (col)

The <col> tag is used to define attributes for one or more columns in an HTML table.

Example:

<table> <colgroup> <col span="2" style="background-color: #f2f2f2;"> </colgroup> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> </tr> <tr> <td>Row 2, Cell 1</td> <td>Row 2, Cell 2</td> </tr> </table>

Styles and Semantics

Style (style)

The <style> tag is used to define inline CSS styles or include external CSS files.

Example:

<style> h1 { color: blue; font-size: 24px; } </style>

Div (div)

The <div> tag is a container that is used to group other HTML elements together.

Example:

<div> <p>This is a div container.</p> <p>It can contain other elements.</p> </div>

Span (span)

The <span> tag is an inline container used to group and style a specific section of text or inline elements.

Example:

<p>This is a <span style="color: blue;">blue</span> text.</p>

Header (header)

The <header> tag is used to define the header section of a document or a section.

Example:

<header> <h1>Welcome to our website!</h1> <p>This is the header section.</p> </header>

Footer (footer)

The <footer> tag is used to define the footer of a document or a section. It typically contains information about the author, copyright notice, links to related documents, or other relevant information.

Example:

<footer> <p>This is the footer of the document.</p> </footer>

Main (main)

The <main> tag is used to define the main content of a document. It represents the main content area and should be unique within the document.

Example:

<main> <h1>Main Content</h1> <p>This is the main section of the page.</p> </main>

Section (section)

The <section> tag is used to define a section or a group of related content within a document.

Example:

<section> <h2>Section Heading</h2> <p>This is a section of the page.</p> </section>

Article (article)

The <article> tag is used to define an independent, self-contained piece of content within a document, such as a blog post, a news article, or a forum post.

Example:

<article> <h2>Article Title</h2> <p>This is an article.</p> </article>

Aside (aside)

The <aside> tag is used to define content that is related to the main content but can be considered separate from it. It can be used for sidebars, pull quotes, or other supporting content.

Example:

<aside> <h3>Sidebar</h3> <p>This is a sidebar section.</p> </aside>

Details (details)

The <details> tag is used to create a disclosure widget that can be toggled to show or hide additional content. It is typically used to create expandable sections with a summary heading.

Example:

<details> <summary>Click me to toggle the details</summary> <p>Hidden details content.</p> </details>

Dialog (dialog)

The <dialog> tag is used to create a modal or a dialog box within a web page. It is typically used for interactive prompts or messages.

Example:

<dialog open> <p>This is a dialog box.</p> <button>Close</button> </dialog>

Summary (summary)

The <summary> tag is used to define a visible heading or summary for the <details> element. It is clicked to toggle the visibility of the details content.

Example:

<details> <summary>Click me to toggle the details</summary> <p>Hidden details content.</p> </details>

Data (data)

The <data> tag is used to define a machine-readable value within the content. It can be used for annotations or providing additional data related to the displayed content.

Example:

<p>The average temperature is <data value="23">23</data>°C.</p>

Meta Info

Head (head)

The <head> tag is used to define the head section of an HTML document. It contains meta-information about the document, such as the title, links to external stylesheets and scripts, and other metadata.

Example:

<head> <title>My Web Page</title> <meta charset="UTF-8"> <link rel="stylesheet" href="styles.css"> <script src="script.js"></script> </head>

Meta (meta)

The <meta> tag is used to provide metadata about an HTML document. It includes information such as character encoding, viewport settings, authorship, and keywords for search engines.

Example:

<meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="This is my website">

Base (base)

The <base> tag specifies the base URL and target for all relative URLs in a document. It is used inside the head section and helps resolve relative links correctly.

Example:

<base href="https://www.example.com/" target="_blank">

Basefont (basefont)

The <basefont> tag is used to specify the base font size, color, and face for a document. However, it is deprecated in HTML5, and its usage is not recommended. CSS should be used to style the document instead.

Example:

<basefont size="4" color="red" face="Arial">

Programming

Script (script)

The <script> tag is used to embed or reference JavaScript code within an HTML document. It can be placed in the head or body section of the document.

Example:

<script src="script.js"></script> <script> // JavaScript code goes here </script>

Noscript (noscript)

The <noscript> tag is used to provide alternative content to be displayed when a browser does not support JavaScript or has JavaScript disabled.

Example:

<noscript> <p>JavaScript is disabled in your browser. Please enable it to view this website.</p> </noscript>

Applet (applet)

The <applet> tag is used to embed Java applets into an HTML document. However, Java applets are no longer supported in most modern browsers.

Example:

<applet code="MyApplet.class" width="300" height="200"> Applet content goes here </applet>

Embed (embed)

The <embed> tag is used to embed various media types, such as audio, video, or interactive content, into an HTML document. It is typically used for browser plug-ins.

Example:

<embed src="video.mp4" type="video/mp4" width="640" height="480">

Object (object)

The <object> tag is used to embed or reference external resources, such as multimedia files or plugins, into an HTML document. It provides fallback content for unsupported browsers.

Example:

<object data="file.pdf" type="application/pdf" width="500" height="600"> Alternative content goes here </object>

Param (param)

The <param> tag is used to define parameters for objects or applets embedded in an HTML document using the <object> or <applet> tag.

Example:

<object data="file.swf"> <param name="autoplay" value="true"> <param name="loop" value="false"> </object>

Formatting

Abbreviation (abbr)

The <abbr> tag is used to define an abbreviation or acronym.

Example:

<p>I work at <abbr title="OpenAI">AI</abbr> company.</p>

Address (address)

The <address> tag is used to define contact information for the author or owner of a document or an article.

Example:

<address> Written by John Doe.<br> Visit us at: <a href="mailto:john@example.com">john@example.com</a><br> 1234 Main Street, Anytown, USA </address>

Bold (b)

The <b> tag is used to make text bold.

Example:

<p>This is <b>bold</b> text.</p>

Bidirectional Override (bdi)

The <bdi> tag is used to isolate a part of text that might be formatted in a different direction from other text outside it.

Example:

<p>This is <bdi>مرحبا بك</bdi> in Arabic.</p>

Bidirectional Override (bdo)

The <bdo> tag is used to override the current text direction.

Example:

<p>This is <bdo dir="rtl">text in RTL direction</bdo>.</p>

Blockquote (blockquote)

The <blockquote> tag is used to specify a section of quoted text.

Example:

<blockquote> <p>This is a quote.</p> <footer>- John Doe</footer> </blockquote>

Citation (cite)

The <cite> tag is used to define the title of a creative work, such as a book, movie, or song.

Example:

<p>I read an interesting article on <cite>example.com</cite>.</p>

Code (code)

The <code> tag is used to define a piece of computer code.

Example:

<p>This is a <code>code</code> example.</p>

Deleted Text (del)

The <del> tag is used to represent deleted or removed text from a document.

Example:

<p>This is <del>deleted</del> text.</p>

Definition (dfn)

The <dfn> tag is used to define the defining instance of a term.

Example:

<p>The <dfn>HTML</dfn> stands for Hypertext Markup Language.</p>

Emphasis (em)

The <em> tag is used to emphasize or highlight text.

Example:

<p>This is <em>emphasized</em> text.</p>

Italic (i)

The <i> tag is used to make text italic.

Example:

<p>This is <i>italic</i> text.</p>

Inserted Text (ins)

The <ins> tag is used to represent inserted text in a document.

Example:

<p>This is <ins>inserted</ins> text.</p>

Keyboard Input (kbd)

The <kbd> tag is used to represent user keyboard input.

Example:

<p>Press <kbd>Ctrl+C</kbd> to copy the selected text.</p>

Marked Text (mark)

The <mark> tag is used to highlight or mark text in a document.

Example:

<p>This is <mark>highlighted</mark> text.</p>

Meter (meter)

The <meter> tag is used to measure data within a specified range.

Example:

<p>Progress: <meter value="75" min="0" max="100">75%</meter></p>

Preformatted Text (pre)

The <pre> tag is used to preserve whitespace and line breaks in the text.

Example:

<pre>This is preformatted text with multiple spaces and line breaks.</pre>

Progress (progress)

The <progress> tag is used to represent the progress of a task.

Example:

<p>Loading: <progress value="60" max="100">60%</progress></p>

Quotation (q)

The <q> tag is used to enclose short inline quotations.

Example:

<p>He said, <q>Hello!</q></p>

Ruby Parenthesis (rp)

The <rp> tag is used to provide parentheses around the ruby text component of a ruby annotation.

Example:

<ruby>漢字 <rp>(</rp> かんじ <rp>)</rp></ruby>

Ruby Text (rt)

The <rt> tag is used to specify the pronunciation of characters presented in a ruby annotation.

Example:

<ruby>漢字 <rp>(</rp> かんじ <rt>Kanji</rt> <rp>)</rp></ruby>

Ruby Annotation (ruby)

The <ruby> tag is used to specify a ruby annotation, which is used for phonetic transcription or annotations of East Asian characters.

Example:

<ruby>漢字 <rp>(</rp> かんじ <rt>Kanji</rt> <rp>)</rp></ruby>

Strikethrough (s)

The <s> tag is used to represent text that is no longer correct or relevant.

Example:

<p>This is <s>strikethrough</s> text.</p>

Sample Output (samp)

The <samp> tag is used to represent sample output or computer code output.

Example:

<p>The output is: <samp>Hello, World!</samp></p>

Small Text (small)

The <small> tag is used to represent smaller text or fine print.

Example:

<p>This is some <small>small text</small> in a paragraph.</p>

Strong Text (strong)

The <strong> tag is used to highlight or emphasize important text.

Example:

<p>This is <strong>strong</strong> text.</p>

Subscript (sub)

The <sub> tag is used to render text in a subscript style.

Example:

<p>H<sub>2</sub>O</p>

Superscript (sup)

The <sup> tag is used to render text in a superscript style.

Example:

<p>2<sup>nd</sup></p>

Template (template)

The <template> tag is used to define reusable content that can be cloned and inserted in the document by JavaScript.

Example:

<template id="myTemplate"> <p>This is a template.</p> </template>

Time (time)

The <time> tag is used to represent a specific date and/or time.

Example:

<p>The event starts at <time datetime="2023-06-01T18:00">6:00 PM</time>.</p>

Underline (u)

The <u> tag is used to underline text.

Example:

<p>This is some <u>underlined text</u> in a paragraph.</p>

Variable (var)

The <var> tag is used to represent the name of a variable in computer code or a mathematical expression.

Example:

<p>Let x be the variable.</p>

Word Break Opportunity (wbr)

The <wbr> tag is used to define a possible line break opportunity within a word.

Example:

<p>This is a longword that may break <wbr>into multiple lines.</p>
